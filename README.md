### SAME-GAMES-APP

Data comes from [RAWG api](https://rawg.io/apidocs)

- [x] Svelte
- [x] Vue
- [x] React
- [ ] Angular
- [ ] Lit
- [ ] Elm
- [ ] Vanilla

Maybe more frameworks in the future?
