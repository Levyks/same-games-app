## SAME-GAMES-APP

Making the same listing of games web app in several different frameworks

Data comes from [RAWG api](https://rawg.io/apidocs)

- [x] Svelte
- [x] Vue
- [x] React
- [x] Elm
- [ ] Angular
- [ ] Lit
- [ ] Vanilla
